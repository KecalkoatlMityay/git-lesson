  • git clone - путь to repository - копирование проекта на гит и так же скачивание актуальной версии с гита к себе
  • git status - статус текущего локального репозитория
  • git add path-to-file - добавить новый ФАЙЛ под контроль гита
  • git commit -am 'added new file index.html' - зафиксировать изменения и дать им комментарий
  • git push - отправляет все изменения на сервак github
  • git pull - закидывает все актуальные файлы с гитхаба на локальный гит
